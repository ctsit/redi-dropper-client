What is RED-I Dropper?

RED-I Dropper is a web-based application used to assist in the secure transport of data from one location to another. RED-I Dropper stores raw MRI and PET images that are tagged with metadata that is relevant to image collection date, type and size. RED-I Dropper is accessible over encrypted channel between participating study sites. The 1Florida ADRC uses RED-I Dropper to share images between multiple analysis sites for collaboration and research.

Features

  • Unlimited image uploads

    • Upload as many files as you need! There are no constraints set on the number of files you upload.


  • Multi and single file downloads

    • If you want to download multiple PET or MRI images for your research, you now have the ability to do so.


  • Filtered downloads by visit date and image type

    • Options to select and filter only the files pertinent to you. No unnecessary downloads!

How to sign up?

In order to sign up for access to RED-I Dropper, the first step is to create an account. To get started please email support@1floridaadrc.org with your first and last name. Someone from the 1Florida ADRC will then contact you to guide you through the additional steps for access.

Are you interested in implementing RED-I Dropper at your own institution? RED-I Dropper is open source and customizable to meet your needs. Check out the source code on our Github repository.

How can you help?

Dropper is constantly undergoing improvement and we want to see it used in real world applications. If you are interested in seeing if Dropper can fit your research needs please contact us at info@1floridaadrc.org.
